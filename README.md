# Mortgage Top-Up Impact Calculator

A financial planning tool designed to visualize how extra mortgage repayments (top-ups) reduce both your loan term and the total interest paid.

## ✨ Features
- **Amortization Simulation**: Real-time calculation of interest decay based on extra contributions.
- **Flexible Frequencies**: Support for weekly, fortnightly, monthly, or annual top-ups.
- **Savings Insight**: Instantly see exactly how much interest you save over the life of the loan.
- **Payment Breakdown**: Shows a comparison table for different payment schedules.
- **Premium UI**: Matches the modern aesthetic of the Premium Unit Converter and Scientific Calculator.

## 🧮 How it Works
The tool calculates a baseline standard payment for your loan. It then simulates the loan balance month-over-month, adding your top-up amount to the monthly payment, to determine the exact month the balance hits zero.

## 🚀 Installation
1. Save the code as `mortgage.html`.
2. Open it in any modern web browser.
